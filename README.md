## ğŸš€ Getting Started

To run this project locally, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/AI-Navigator.git](https://github.com/YOUR_USERNAME/AI-Navigator.git)
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd AI-Navigator
    ```
3.  **Launch the App:**
    * Since this project uses ES6 Modules, you need a local server.
    * If using VS Code, simply right-click `index.html` and select **"Open with Live Server"**.
4.  **Configuration:**
    * On the first launch, the app will ask for your **Gemini API Key**.
    * You can get a free key from [Google AI Studio](https://aistudio.google.com/).
